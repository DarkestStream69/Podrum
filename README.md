# PodrumR3

## Introduction
Podrum is a server software for Minecraft Bedrock Edition written in C and aims to be fast, stable and secure.

This project is still in early development.

## Building
Linux/MacOS:
```
cmake CMakeLists.txt
make
```

Windows:
```
cmake CMakeLists.txt
msbuild PodrumR3.sln
```
